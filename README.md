# CodeBreaker

## Project description

SwiftUI App based on the course [CS193p - Stanford University](https://cs193p.stanford.edu/)

## Game Overview

CodeBreaker is an engaging puzzle game where you must guess a secret peg code. Each guess provides visual feedback, helping you narrow down the correct combination through logical deduction.

## How to Play

1. **The Secret Code**: The game generates a random peg code at the start (displayed at the top)
2. **Make Your Guess**: Tap on any peg in the guess row to cycle through available colors
3. **Submit Your Guess**: Tap the large "Guess" button to check your guess against the secret code
4. **Analyze Feedback**: After each guess, you'll receive feedback markers:
   - **Filled circles**: Exact matches (correct color in correct position)
   - **Thick outlined circles**: Inexact matches (correct color in wrong position)
5. **Review History**: Scroll through previous attempts with their feedback
6. **Win the Game**: When all 4 markers are filled, you've successfully cracked the code!
